File "Data Tool Set Composition.xlsx":
- TL refers to Tool Set (e.g., TL1): a list of required individual tools to process an operation (e.g., TA08122, TA00071, TA02186, etc.).
- TA (e.g., in tool TA08122) refers to Tool Assembly (assembly of tool holder and cutting tool).

File "Data Jobs.xlsx":
- Each [Job] consists of a product type [Item].
- Each [Job] requires a tool set [TL] that contains individual tools for processing (see file "Data Tool Set Composition.xlsx").
- The [Processing Time] resembles the processing time of an operation of a job. When a job has two operations (reentrance), they have the same processing time.
